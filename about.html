Scholarship Management System ----------------------------- Description: This project is a web-based Scholarship Management System designed to streamline and digitize the process of managing student scholarships. It allows students to register, apply for scholarships, and login to track their application status. Administrators can manage applications, review eligibility, and approve or reject requests. Features: - Student registration and login - Secure login system with password protection - Application form submission for scholarships - Admin panel for reviewing and managing applications - Clean and responsive user interface using HTML & CSS Technologies Used: - Frontend: HTML, CSS - Version Control: Git Directory Structure: - index.html → Home or landing page - login.html → Student login page - register.html → Student registration form - dashboard.html → Student or admin dashboard - about.txt → This file (project description) - css/ → Stylesheets How to Use: 1.
